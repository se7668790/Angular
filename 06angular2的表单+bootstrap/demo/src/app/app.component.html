由于本文使用了BootStrap框架定义UI，所以需要：
  1. npm install --save bootstrap
  2. 在页面引入node_modules/bootstrap/dist/css/bootstrap.min.css

ng2 的表单和 ng1 类似, 也是用 control 概念来做操作, 当然也有一些地方不同 最大的特点是它把表单区分为 template drive and model drive

template drive 和 ng1 很像, 就是通过指令来创建表单的 control 来加以操作.

model drive 则是直接在 component 内生成 control 然后再绑定到模板上去.

template drive 的好处是写得少，简单, 适合用于简单的表单
